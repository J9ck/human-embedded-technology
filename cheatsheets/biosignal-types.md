## Typical Sampling Rates

- **EEG**: 250-500 Hz (adequate), 1-2 kHz (research)
- **EMG**: 1-2 kHz (surface), 10-30 kHz (needle)
- **ECG**: 250-500 Hz (clinical), 1 kHz (research)
- **Neural Spikes**: 20-30 kHz

---

## Electrode Types

### Non-Invasive
- **Ag/AgCl**: Standard wet electrodes (low impedance)
- **Dry electrodes**: No gel required (higher impedance)
- **Active electrodes**: Built-in amplifier (noise reduction)

### Invasive
- **Utah array**: 100-electrode silicon microelectrode array
- **Flexible arrays**: Polyimide-based, conform to brain
- **Depth electrodes**: Deep brain structures (DBS)
